== Description ==

CBX Latest Tweets plugin helps to show tweets from twitter account. support multiple account using comma.
Ready made 6+ layouts to display tweets as fits for your theme and website.
Supports Shortcode, Widgets, Gutenberg Block, Elementor Widget, WPBakery Widget


[youtube https://www.youtube.com/watch?v=TJY6aQJbZtY]

1. There are multiple ways to install a wordpress plugin https://codex.wordpress.org/Managing_Plugins
2. Activate the plugin through the 'Plugins' menu in WordPress
3. You'll now see a menu called "CBX Latest Tweets" under 'Setting' in left menu of word, start from there.
4. This plugin has shortcode, widget or direct function to call.
